# MongoDB Crash Course
A comprehensive and hands-on MongoDB crash course designed to help developers learn MongoDB from fundamentals to practical usage using real-world examples. This repository contains all the source code, examples, and practice files used in the **MongoDB Crash Course 20**, focusing on modern MongoDB development with JavaScript.

## What You Will Learn

- MongoDB fundamentals and core concepts  
- Installing and setting up MongoDB  
- Database, collections, and documents  
- CRUD operations (Create, Read, Update, Delete)  
- Querying and filtering data  
- Aggregation framework  
- Indexes and performance optimization  
- Working with MongoDB using JavaScript  
- Best practices for real-world applications

## Getting Started

### Prerequisites

Make sure you have the following installed:

* MongoDB (Community Edition)
* Node.js (LTS recommended)
* MongoDB Compass (optional but recommended)

### Clone the Repository

```bash
git clone https://github.com/Md-Emon-Hasan/MongoDB-Crash-Course.git
cd MongoDB-Crash-Course.git
```

### Run MongoDB

```bash
mongod
```

or use MongoDB Compass / MongoDB Atlas as preferred.
